# ADR-0001: Project vision

**Status:** Accepted

**Date:** 2026-06-29

# Context

The Columbia County Democrats currently host their website on Squarespace and use Mailchimp for email. They use other systems including NGP Van, Action Network and Mobilize as well as spreadsheets to manage contact relationship information.

# Decision

This project aims to build an open-source CRM platform for county Democratic parties using WordPress and CiviCRM. Columbia County Democrats (Columbia County, Wisconsin, aka CoCoDems) will serve as the prototype. The long-term goal is a reusable platform that integrates volunteer management, email, donations, events, and websites while interoperating with NGP VAN and Action Network.

# Rationale

In addition to letting us get experience with vibe coding, the project will enable CoCoDems to consolidate multiple existing IT systems, including the existing Squarespace website, Mailchimp, and contact information that is currently being stored in multiple spreadsheets. It will also manage data about county elections, elected offices, officeholders and jurisdictions including cities, towns, villages, school districts.

The project will either integrate with or supplant some existing Democratic Party IT resources including NGP Van, Action Network and Mobilize. Initially, however, it will exist alongside those resources.

If successful, the project may also become a prototype for CRMs serving other county Democratic parties.
