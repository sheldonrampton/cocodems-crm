# ADR-0003: Use WordPress + CiviCRM Instead of Squarespace

**Status:** Accepted

**Date:** 2026-06-29

# Context

The Columbia County Democrats currently host their website on Squarespace and use Mailchimp for email.

We want a unified system that supports:

* website
* volunteer management
* donations
* events
* email
* CRM

# Decision

Use WordPress integrated with CiviCRM.

# Rationale

## Advantages:

* Open source
* Lower recurring cost
* Deep CiviCRM integration
* Full control over data
* Extensive plugin ecosystem
* Can replace both Squarespace and Mailchimp

## Disadvantages:

* More technical administration
* More maintenance

# Consequences

The project requires Linux hosting and technical administration but gains long-term flexibility and lower operating costs.
